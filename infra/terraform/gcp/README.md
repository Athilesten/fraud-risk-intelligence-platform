# GCP Terraform Blueprint

This is a documentation-first placeholder for a GCP production deployment.

Recommended services:

- Frontend: Cloud Storage + Cloud CDN
- API: Cloud Run
- Database: Cloud SQL for PostgreSQL
- Streaming: Confluent Cloud or Pub/Sub alternative
- Data lake: Google Cloud Storage
- Spark: Dataproc Serverless
- Secrets: Secret Manager
- Monitoring: Cloud Monitoring + Grafana

No real credentials or projects are required for the local demo.
