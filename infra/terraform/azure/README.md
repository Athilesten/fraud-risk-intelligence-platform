# Azure Terraform Blueprint

This is a documentation-first placeholder for an Azure production deployment.

Recommended services:

- Frontend: Azure Static Web Apps
- API: Azure Container Apps
- Database: Azure Database for PostgreSQL
- Streaming: Confluent Cloud or Event Hubs Kafka API
- Data lake: ADLS Gen2
- Spark: Azure Databricks
- Secrets: Azure Key Vault
- Monitoring: Azure Monitor + Grafana

No real credentials or subscriptions are required for the local demo.
